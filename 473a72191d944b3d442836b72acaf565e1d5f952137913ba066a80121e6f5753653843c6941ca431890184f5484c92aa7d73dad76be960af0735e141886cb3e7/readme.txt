Google From式のポスター追加マニュアル

【ポスター追加を行う時期について】
2年次の2月頃に一度完成させて提出してもらうときがありますが、その後ポスターに変更が加わる可能性が高いので、祥雲祭前の3年次のポスターPDFの"最終"提出と同時に行ってください。


【追加作業の負担、必要時間について】
・生徒にはGoogle Fromに研究ポスターの情報を入力してもらう作業が加わるだけなので、大きな負担にはならないと思います。
・追加の作業を担当する方も、15分程度で終わる作業量だと思います。


【大まかな流れ】
・スプレッドシートとフォームにアクセスできることを確認
・3年次全員にフォームを共有
・探究ⅢとしてのポスターPDFの最終提出をするとき、各研究班から1人だけがフォームを入力
・締め切り後、スプレッドシートを開いてJSONで出力をクリックしてdata.jsを生成
・data.jsとPDFを回生フォルダー直下に配置
・index.htmlとscript.jsにパスを追加
↓余裕があれば
・その後3年次全員にShounPosterFinderを使って自分の班の研究ポスターが見れるか確認してもらう
・問題があれば担当者へ報告してもらう
・data.jsの中と、ポスターPDFのファイル名を確認して修正


【詳しい追加方法】
「Shoun Poster Finder」という名前の共有ドライブがあるので、そこへのアクセス権をもらってください。

祥雲祭のポスターを提出する際、3年次に以下のGoogle Formのリンクを送ってください。
https://forms.gle/bV8VpNP5vtusUsub9

解答が出そろったら、Formとリンクされてるスプレッドシートを開いてください。以下がそのスプレッドシートのリンクです。
https://docs.google.com/spreadsheets/d/18dir-8YS7ZA4GCp7vKAAolhljMyLvJrRiNXBn7nNhW4/edit

スプレッドシートの読み込みが終わったら、ヘッダー(一番上左)の右の方から｢出力｣を探します。
見つけたらクリックして｢JSONで出力｣をクリックします。

初めての場合、｢権限が必要です｣などといったダイアログが表示されますが、｢続行｣などのボタンをクリックして進めてください。
ウィンドウが開いてGoogleアカウントのログイン画面になると思います。
学校のGoogleアカウントにログインし、すべての権限を許可してください。
本来要求する権限は｢スプレッドシートの参照･作成・削除など｣と｢Googleドライブの参照・作成・削除など｣ですが、それ以外が要求された場合はコードが書き換えられている恐れがあるので、それ以上は触らずに管理者に報告してください。

「何回生かを半角で入力してください。」のダイアログが表示されたら、例の通りに入力します。
22回生の分のデータを登録するのであれば｢22｣と半角で入力します。

入力したら｢OK｣などのボタンを探してクリックします。
一旦表示が消え、しばらくすると「成功」といったダイアログが表示されます。
スプレッドシートが保存されているフォルダーに｢data.js｣が作られています。
この「data.js」にスプレッドシートのデータが詰まっています。

｢data.js｣をSPFの中の「PDF」の中の各回生のフォルダーに移します。
例）SPF\PDF\22\data.js
同じフォルダー内にポスターPDFも入れます。
例）SPF\PDF\22\poster.pdf
※20回生まではジャンルごとにフォルダ分けしてPDFを保存していますが、21回生以降はジャンルフォルダーを廃止しています。


data.jsとポスターPDFの配置が終わったら、index.htmlとscript.jsを編集します。

まず、index.htmlについて。
「<script src="PDF/zz/data.js"></script>　(zzは半角数字)」がたくさん並んでる箇所を探します。
見つけたら、たくさん並んでる一番下に、上と同じような感じで追加します。一つ上のやつをコピペして、数字だけ書き換える方法をオススメします。
zzの部分は、先ほど配置したdata.jsが保存されているフォルダー名にしてください。

例）
<script src="PDF/20/data.js"></script>
<script src="PDF/21/data.js"></script>
<script src="PDF/22/data.js"></script>      ←追加

次に｢<label><input type="checkbox" class="generation-filter" value="xx">xx</label>  (xxスプレッドシートで入力した半角数字=回生)｣がたくさん並んでる箇所を探します。
見つけたら、たくさん並んでる一番下に、上と同じような感じで追加します。一つ上のやつをコピペして、数字だけ書き換える方法をオススメします。

例）
<label><input type="checkbox" class="generation-filter" value="20">20</label>
<label><input type="checkbox" class="generation-filter" value="21">21</label>
<label><input type="checkbox" class="generation-filter" value="22">22</label>   ←追加
                

index.htmlの編集が終わったらscript.jsの編集に移ります。
「if (typeof pdfDataXX !== 'undefined') pdfData = pdfData.concat(pdfDataXX); (スプレッドシートで入力した半角数字=回生　２か所)」がたくさん並んでる箇所を探します。
見つけたら、たくさん並んでる一番下に、上と同じような感じで追加します。一つ上のやつをコピペして、数字だけ書き換える方法をオススメします。

例）
if (typeof pdfData20 !== 'undefined') pdfData = pdfData.concat(pdfData20);
if (typeof pdfData21 !== 'undefined') pdfData = pdfData.concat(pdfData21);
if (typeof pdfData22 !== 'undefined') pdfData = pdfData.concat(pdfData22);  ←追加


以上でポスター追加作業は完了です。


最後に、3年次に自分のポスターが見れるか確認してもらってください。
見れなかった場合、大抵の場合はFormでの入力ミスなので、data.jsを開いて該当箇所を正しく修正すれば直ります。
稀に、一つのエラーのせいですべてが表示されなくなる場合がありますが、落ち着いて対処すれば大丈夫なので落ち着きましょう。

回答が集まったシートの内容をすべて消すか、フォームをリンクし直して新しいシートを追加すれば、上記フォームとスプレッドシートを来年も使いまわせます。
!!!スップレドシートには｢Google Apps Script (GAS)｣というプログラムが施されています。スプレッドシートを作り直す場合はGASをコピペしてください。不安がある場合は使いまわすことをおすすめします。
